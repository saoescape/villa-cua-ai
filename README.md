# Villa Cua AI Concierge

Progetto completo React + Vite + TypeScript per GitHub Pages.

## Pubblicazione

1. Estrai lo ZIP.
2. Apri il repository GitHub `villa-cua-ai`.
3. Elimina i vecchi file della demo, oppure sostituiscili caricando tutti i file di questo progetto.
4. Mantieni la struttura delle cartelle, compresa `.github/workflows/deploy.yml`.
5. Esegui il commit sul branch `main`.
6. In `Settings > Pages`, seleziona **GitHub Actions**.
7. Apri la scheda `Actions` e attendi il segno verde.
